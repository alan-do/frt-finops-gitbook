## 2. Chi tiết Lộ trình (Milestones)

<Steps>

### Bước 1: Data Pipeline Foundation (Tháng 6/2026)
**Mục tiêu:** Gom dữ liệu từ các "đảo" (Silos) về một nguồn duy nhất.
- Kéo dữ liệu Billing từ AWS Cost Explorer.
- Lấy API Usage từ OpenAI / Azure / Anthropic.
- Chuẩn hóa dữ liệu và lưu trữ tập trung (Centralized Data Lake).

### Bước 2: Alerting & Cost Allocation (Tháng 7/2026)
**Mục tiêu:** Gắn nhãn chi phí và cảnh báo tức thời.
- Xây dựng hệ thống Tagging để chia tiền (Cost Allocation) cho từng Product Team.
- Thiết lập Webhook cảnh báo lên **Slack** ngay khi chi phí tăng đột biến (Anomaly Detection).

### Bước 3: Real-time Dashboard (Tháng 8/2026)
**Mục tiêu:** Trực quan hóa dữ liệu cho Management Team.
- Dựng Grafana/Metabase Dashboard cho CTO và Product Owners.
- Thêm tính năng Filter theo thời gian, theo team, theo môi trường (Dev/Staging/Prod).

### Bước 4: GenAI Automated Reporting (Tháng 9/2026)
**Mục tiêu:** Để AI tự giải thích nguyên nhân tăng/giảm chi phí.
- Tích hợp ChatGPT/Claude để đọc dữ liệu chi phí và viết báo cáo phân tích tự động.
- Trả lời các câu hỏi: *"Tại sao team A tiêu tốn nhiều token OpenAI tuần này?"*

</Steps>
