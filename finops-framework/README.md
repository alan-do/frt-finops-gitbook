# FinOps Framework tại FRT

Phần này sẽ map (ánh xạ) các khái niệm cốt lõi của **FinOps Framework** (từ FinOps Foundation) vào bối cảnh thực tế của Ban Công nghệ FRT.

Chúng ta sẽ tìm hiểu về các nguyên tắc cốt lõi hướng dẫn mọi hoạt động tối ưu chi phí, cũng như vòng lặp 3 giai đoạn (Phases) của FinOps: **Inform (Thông tin) -> Optimize (Tối ưu) -> Operate (Vận hành)**.
